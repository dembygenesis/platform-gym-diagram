# Platform Gym — System Design (learn-fast edition)

A single self-contained, mobile-friendly HTML page that maps the whole
UpCloud + k3s + GitOps platform gym: the deploy loop, bootstrap ordering,
request path, secret flow, namespace map, and the failure-mode "scars".

No build, no CDN, no JS framework — just open `index.html` (or the Pages link).

**Live:** https://dembygenesis.github.io/platform-gym-diagram/
